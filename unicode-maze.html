Unicode Maze Game

This is the a simple maze game that is made out of unicode characters. The one playable here is written in JavaScript because it made implementing it into a webpage much easier. However, the original version is written in Python because it's what I was learning at the time.
Also, this has only been tested to work on Chrome on Windows 7. Other browsers may render the unicode differently and make this unplayable.
